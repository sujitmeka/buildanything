# BrokenSpec

## Overview

BrokenSpec is a project-management tool aimed at independent contractors. It needs to feel calm, professional, and easy to learn for non-technical users — sole proprietors who track invoices, hours, and client communication in one place. The brand voice is approachable but not playful; trustworthy but not corporate.

The product needs to support contractors who switch between job sites and home offices, so dark mode and high-contrast variants are first-class. Pricing is a single tier — there is no "enterprise" SKU and no marketing-page funnel beyond the homepage.
