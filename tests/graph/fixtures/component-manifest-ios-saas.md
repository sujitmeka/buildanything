## SwiftUI version pin

- Minimum iOS deployment target: iOS 18 (Liquid Glass variants gated on iOS 26+ runtime check)
- Xcode: 26.3 (per project lock)
- No third-party UI libraries — SwiftUI native only per protocol §9

## Cleanup contract

Per protocol §9.3, the iOS implementer MUST use the named SwiftUI primitive for [HG] slots. UIKit fallbacks, third-party UI libraries, and hand-rolled Canvas reimplementations are forbidden — the iOS cleanup agent reverts them.
